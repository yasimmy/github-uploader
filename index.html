Как получить GitHub токен?

1

Откройте настройки GitHub

Перейдите на github.com/settings/tokens

2

Создайте новый токен

Нажмите "Generate new token" → "Generate new token (classic)"

3

Настройте права доступа

Выберите следующие права:

  • ✓ repo (полный доступ к репозиториям)
  • ✓ workflow (если используете GitHub Actions)
4

Скопируйте токен

Нажмите "Generate token" и скопируйте созданный токен

⚠️ Токен показывается только один раз! Сохраните его в безопасном месте.
