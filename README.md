# GitHub Загрузчик

Desktop приложение для загрузки проектов на GitHub с красивым темным интерфейсом.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

## Как использовать

### 1. Получение GitHub токена

1. Откройте [github.com/settings/tokens](https://github.com/settings/tokens)
2. Нажмите "Generate new token" → "Generate new token (classic)"
3. Выберите права доступа:
   - ✓ **repo** (полный доступ к репозиториям)
   - ✓ **workflow** (опционально, для GitHub Actions)
4. Нажмите "Generate token" и скопируйте токен
5. ⚠️ Токен показывается только один раз! Сохраните его.

### 2. Использование приложения

1. Запустите приложение
2. Вставьте токен (сохранится автоматически)
3. Выберите папку с проектом
4. Заполните:
   - Название репозитория
   - Описание (опционально)
   - Сообщение коммита (по умолчанию "Первый коммит")
   - Приватность репозитория
5. Нажмите "Загрузить на GitHub"
